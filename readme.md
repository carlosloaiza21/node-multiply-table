# Notas:

Commands to run by command line:

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     is the table number                         [number] [required]
  -l, --list     show the table on console            [boolean] [default: false]
  -t, --To       limit                                    [number] [default: 10]
```
